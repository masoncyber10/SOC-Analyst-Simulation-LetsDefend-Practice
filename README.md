# SOC-Analyst-Simulation-LetsDefend-Practice
<p>This repository documents my activities and learnings from the SOC Analyst simulation on the LetsDefend platform. The simulation involved monitoring security alerts, analyzing potential threats, responding to incidents, and improving detection rules to mimic real-world SOC operations.</p>
<p1>The goal of the soc analyst simulation is to practice the role of a soc analyst through hands-on, interview-ready experience with monitoring and analyzing network and endpoint security, investigate incidents with medium to high alerts, respond to security events, and improve alert accuracy and reduce false positives, and to document every step.</p1>
